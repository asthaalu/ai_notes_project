# AI Notes Explainer & Doubt Solver

This project is a Generative AI-based application that helps students summarize notes and solve doubts.

## Features
- Notes Summarization
- Doubt Solving using AI
- PDF Summarizer
- Interactive Streamlit UI

## Technologies Used
- Python
- Streamlit
- Transformers
- Sentence-Transformers

## How to Run
pip install -r requirements.txt
streamlit run app.py
